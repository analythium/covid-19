Methods About

Data shown here reflect the current number of cases (confirmed – recovered – deaths) by region. We analyzed the time series of current cases starting from the 1st day of confirmed cases in that region. We used exponential smoothing state space model to fit a time series model to the natural logarithm transformed data. We then forecast based on this models providing the point predictions and 95% confidence limits. This forecast is uncertain and might change based on new data. It is intended to show short term trend in the number of cases assuming all other factors affecting the number of cases, such as intervention measures, remain unchanged.

The Website relies upon publicly available and daily updated data from multiple sources as discribed on the JHU CSSE page. The data behind the plot can be accessed through an API descrined on the Methods page.
